# Python-Tetris
Tetris game made with the Python package pygame.


# Controls

Piece navigation: 
- - Move Left (Left arrow key / 'a' key)
- - Move Right (Right arrow key / 'd' key)
- - Move Down (Down arrow key / 's' key)

Piece Rotation:
- - Rotate clockwise ('x' key)
- - Rotate counter-clockwise ('z' key)
